# FFmpeg Lab

یک محیط آزمایشگاهی تعاملی برای اجرای سریع دستورات FFmpeg است.

این ابزار با یک رابط HTML ساده، به شما کمک می‌کند بدون نیاز به حفظ کردن دستورات پیچیده FFmpeg، عملیات‌های رایج پردازش ویدیو را انجام دهید.

کاربر فقط پارامترها را در رابط وارد می‌کند و ابزار دستور FFmpeg مناسب را تولید می‌کند تا در ترمینال اجرا شود.

این پروژه به‌صورت **Portable Folder Launcher** طراحی شده است؛ یعنی بدون نصب خاصی و فقط با اجرای یک فایل `.bat` آماده استفاده است.

---

# ویژگی‌ها

- رابط کاربری ساده HTML
- تولید خودکار دستورات FFmpeg
- اجرای مستقیم در محیط CMD
- ساختار پوشه‌ای منظم برای ورودی و خروجی
- بدون نیاز به نصب نرم‌افزار اضافی
- قابل حمل (Portable)

---

# ساختار پروژه

پوشه اصلی پروژه به شکل زیر است:

```
FFmpeg-Lab/
│
├─ FFmpeg-Lab.html
│   رابط کاربری اصلی آزمایشگاه
│
├─ start-lab.bat
│   لانچر پروژه که محیط را آماده کرده و HTML را اجرا می‌کند
│
├─ ffmpeg.exe
│   نسخه پرتابل FFmpeg که تمام دستورات توسط آن اجرا می‌شوند
│
├─ input/
│   پوشه فایل‌های ورودی (ویدیو، تصویر، زیرنویس، واترمارک و ...)
│
└─ output/
    پوشه فایل‌های خروجی تولید شده
```

---

# نحوه راه‌اندازی

1. مخزن را دانلود یا Clone کنید:

```
git clone https://github.com/hosseinyariam/FFmpeg-Lab.git
```

یا فایل ZIP را دانلود کنید.

2. وارد پوشه پروژه شوید.

```
FFmpeg-Lab
```

3. فایل زیر را اجرا کنید:

```
start-lab.bat
```

این اسکریپت:

- پوشه‌های `input` و `output` را در صورت نیاز ایجاد می‌کند
- فایل HTML را در مرورگر باز می‌کند
- یک پنجره CMD برای اجرای دستورات FFmpeg آماده نگه می‌دارد

---

# نحوه استفاده

### 1️⃣ قرار دادن فایل‌ها

تمام فایل‌هایی که قرار است پردازش شوند باید داخل پوشه زیر قرار بگیرند:

```
input/
```

مثال:

```
input/video.mp4
input/cover.jpg
input/subtitle.srt
```

---

### 2️⃣ تولید دستور

در صفحه **FFmpeg Lab**:

- ابزار مورد نظر را انتخاب کنید
- پارامترها را وارد کنید
- دستور FFmpeg تولید می‌شود

---

### 3️⃣ اجرای دستور

دستور تولید شده را در پنجره CMD که توسط `start-lab.bat` باز شده است اجرا کنید.

---

### 4️⃣ دریافت خروجی

تمام فایل‌های خروجی در پوشه زیر ذخیره می‌شوند:

```
output/
```

---

# مثال

ساخت ویدیو با کاور ابتدایی:

1. قرار دادن فایل‌ها:

```
input/video.mp4
input/cover.jpg
```

2. تولید دستور از ابزار **Add Cover Intro**

3. اجرای دستور در CMD

خروجی:

```
output/output-intro-3s.mp4
```

# ابزارهای موجود

این آزمایشگاه شامل مجموعه‌ای از ابزارهای کاربردی FFmpeg است، مانند:

- تغییر اندازه ویدیو
- تبدیل فرمت
- افزودن واترمارک
- افزودن زیرنویس
- برش ویدیو
- ادغام ویدیو
- اضافه کردن کاور ابتدایی
- تنظیم bitrate و کیفیت
- استخراج صدا

و چندین عملیات رایج دیگر.

---

# نکات مهم

- نام فایل‌ها را **بدون فاصله یا با نقل قول استفاده کنید**
- تمام ورودی‌ها باید داخل پوشه `input` باشند
- خروجی‌ها همیشه داخل `output` ذخیره می‌شوند
- این ابزار فقط **دستورات FFmpeg را تولید می‌کند** و اجرا توسط CMD انجام می‌شود
